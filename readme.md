# 🎫 BookIt - Easy Event Ticketing Platform

A full-stack event booking application built with a secure database setup to prevent ticket overselling, complete with an attendee marketplace and an organizer dashboard.

---

## ⚡ Quickstart Guide (Run in 2 Simple Steps)

Follow these two steps to spin up the entire application on your computer—including the database, backend server, and frontend website.

### Prerequisites

Make sure you have Docker Desktop open and running on your computer.

### Step 1: Set Up Environment Variables

Create a file named `.env` inside your `backend/` folder and add these configuration lines:

```env
PORT=5000
DATABASE_URL="postgresql://postgres:password123@db:5432/bookit?schema=public"
JWT_SECRET="your_secret_key_here"
NODE_ENV=production
```

### Step 2: Run the App with Docker Compose

Open your terminal at the project root folder (`BookIt2/`) and run this single command:

```bash
docker-compose up --build
```

---

## 📦 What Happens Behind the Scenes? (Database Setup)

You don't need to manually run migrations or seeding commands! The `docker-compose.yml` file is configured to execute everything in a chain automatically as soon as the containers wake up:

- `npx prisma migrate deploy` — This command automatically reads our structural database files and sets up the clean tables inside PostgreSQL.
- `npx prisma db seed` — This command runs our custom script to automatically plant mock data, profiles, and 7 realistic event listings into the database so the app isn't blank on launch.
- `npm run dev` — This turns on the live active development servers.

Once the logs stop scrolling, open your browser to check out the running app layers:

### Frontend Website Layout

```
http://localhost:3000
```

### Backend Live Server API Data

```
http://localhost:5000/api/v1/events
```
